Employee Survey Web-Based

Web Development

A web-based application designed to simplify employee feedback collection and automate result analysis within a company. The system allows employees to fill out surveys easily while providing admins with tools to manage data and monitor participation progress.

Technologies Used: HTML, Tailwind CSS, JavaScript, PHP (Native), MySQL

Libraries: Chart.js, DataTables

Key Features:

  • Admin dashboard with CRUD functionality for question and employee data
  • Data visualization using Chart.js and DataTables
  • Responsive design optimized for both desktop and mobile devices
  • Export survey data to Excel or SVG format for reporting and analysis
  • User page for filling out and submitting surveys
