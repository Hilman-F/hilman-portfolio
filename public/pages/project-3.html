Smart Hydroponic Automation System

Internet of Things

An IoT device designed to automate hydroponic cultivation by monitoring nutrient levels, pH, and temperature in real time. It supports both manual and automatic modes powered by fuzzy logic, ensuring optimal growth conditions. The system also transmits sensor data to a mobile app for easy tracking and management.

Technologies Used: Arduino, C/C++, ESP32, Fuzzy Logic Control

Communication: Firebase

Key Features:

  • Real-Time Monitoring – Measure and transmit live nutrient, pH, and temperature data.
  • Fuzzy Logic Control – Automatically adjust system parameters based on condition changes.
  • System Feedback – Adjusts nutrient dosing dynamically based on sensor feedback.
  • Manual & Auto Modes – Allow users to switch between automatic and manual operation.
  • Data Synchronization – Send sensor data to the mobile app for visualization and tracking.
